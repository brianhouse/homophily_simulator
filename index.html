DENSITY

  • This is a board composed of agents
  • There are two immutable groups, circle and square
  • Each agent also has an "attitude" which is a continuous value

  • Agents want to be surrounded by other agents with similar values on average
  • If there is too large a discrepancy, the agent will be unhappy (marked with a red dot)
  • If an agent is unhappy, it will try to move somewhere where it is happier
  • Agents are also affected by the attitudes of the agents around them, which is turn may affect their happiness and desire to move
